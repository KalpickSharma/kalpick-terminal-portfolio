# Terminal Portfolio (React + Vite + Tailwind)

## Quick start
```bash
npm install
npm run dev
# open the URL from the terminal
```

## Available commands inside the terminal UI
- `help`
- `about`
- `skills`
- `projects`
- `links`
- `contact`
- `test`
- `clear`

## Build
```bash
npm run build
npm run preview
```
